SAMPLE SIZE

The sample size plot shows the number of paired data available by forecast lead time and threshold (if specified). A verification pair comprises a single forecast at a particular lead time and a corresponding observation, i.e. an observation recorded at the same actual time.

The sample size plot is not a verification metric, but may be used for data exploration. For example, it may assist in the selection of thresholds for computing conditional verification metrics, where the aim is to produce statistics with a 'reasonable' sample size, i.e. reasonably low sampling uncertainty.
