SINGLE-VALUED QUANTILE-QUANTILE PLOT

The single-valued quantile-quantile plot compares the order statistics of the observations and corresponding single-valued forecasts. The single-valued forecasts may be derived from a set of ensemble forecasts using a specified function. For example, when using the ensemble mean forecast, the quantile- quantile plot shows the biases in the ensemble mean forecast (as a departure from the diagonal line) by observed quantile.
